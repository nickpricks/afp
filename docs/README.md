# docs

Project documentation root for AFP.

## Key Files

- `ROADMAP.md` -- Full prioritized backlog (P0-P3) for remaining work
- `firebase-setup.md` -- Firebase project configuration and setup instructions
- `getting-started.md` -- Developer onboarding guide

## Subdirectories

- `plans/` -- Implementation plans (Phase 1 and Phase 2 breakdowns)
- `specs/` -- Design specifications and analysis documents
- `revz/` -- Code reviews, audits, and coverage analysis
