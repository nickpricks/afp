# Phase 2f: Theme System Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** Replace the current 7-theme system with 10 finalized themes (drop 3, rename 1, add 6), with distinct font pairings applied via CSS variables, configurable ambient effects per theme, and user-level effect customization in the Profile page.

**Architecture:** Each theme is a CSS file defining custom properties (light + dark variants where applicable), registered in `themes.ts` via `ThemeId` enum + `THEME_DEFINITIONS`. `applyTheme()` handles class switching, font variable injection, and effect class toggling. Effects are CSS animations in `effects.css`, rendered via a `.fx-ambient` container in Layout. Google Fonts loaded via `<link>` in `index.html`.

**Tech Stack:** CSS custom properties, Google Fonts, CSS keyframe animations, React 19, TypeScript, vitest

**Spec:** `docs/specs/2026-04-10-themes-design.md`
**Showcase:** `SAM/design-samples/theme-showcase-all.html` (source of truth for all hex values, fonts, effects)

In `src/themes/themes.ts`, update `applyTheme()` to set font CSS variables:

```ts
export function applyTheme(themeId: ThemeId, colorMode: ColorMode): void {
  const theme = THEME_DEFINITIONS[themeId];
  const root = document.documentElement;
  const newClass = themeClass(themeId);

  if (currentThemeClass) root.classList.remove(currentThemeClass);
  root.classList.add(newClass);
  currentThemeClass = newClass;

  // Apply fonts
  root.style.setProperty('--font-display', `'${theme.fonts.display}', sans-serif`);
  root.style.setProperty('--font-body', `'${theme.fonts.body}', sans-serif`);
  root.style.fontFamily = `'${theme.fonts.body}', sans-serif`;

  if (theme.darkOnly || colorMode === 'dark') {
    root.classList.add('dark');
  } else if (colorMode === 'light') {
    root.classList.remove('dark');
  } else {
    const isDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
    root.classList.toggle('dark', isDark);
  }
}
```
