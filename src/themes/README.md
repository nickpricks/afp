# themes/

CSS custom properties per theme, mapped to Tailwind via `@theme` in index.css.

## Files

- **themes.ts** — `ThemeId` enum, `THEME_DEFINITIONS` record, `themeClass()`, `applyTheme()`, `isValidThemeId()`, and `useActiveThemeId()` hook
- **buttons.css** — Shared button styles across themes
- **effects.css** — Shared visual effects (transitions, animations)
- **family-blue.css** — Family Blue theme (light + dark, default)
- **summit-instrument.css** — Summit Instrument theme (light + dark)
- **corporate-glass.css** — Corporate Glass theme (light + dark)
- **deep-mariana.css** — Deep Mariana theme (dark only)
- **night-city-elevator.css** — Night City: Elevator theme (dark only)
- **night-city-apartment.css** — Night City: Apartment theme (dark only)
- **industrial-furnace.css** — Industrial Furnace theme (dark only)
